# 🛡️ Cybersecurity Incident Dashboard

A data analytics project exploring global cybersecurity threats from 2015 to 2024.

## 📊 Dataset
- **Source:** Kaggle — Global Cybersecurity Threats 2015–2024
- **Size:** 3,000 incidents across 10 features
- **Coverage:** Attack types, target industries, financial losses, affected users

- [x] Phase 1: Environment Setup & Data Sourcing
- [ ] Phase 2: Data Cleaning
- [ ] Phase 3: Exploratory Data Analysis
- [ ] Phase 4: Data Visualization
